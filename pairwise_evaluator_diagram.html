Stage 3 — Global Ranking (across many comparisons)
📐 Bradley-Terry Model
βi = Wi / Σj≠i [ nij / (βi + βj) ]
βi — latent strength of setup i (what we solve for)
Wi — total wins by setup i
nij — number of comparisons between i and j
Iterate MM update until convergence → ranked list of setups by β score.
🎯 Using Agent Confidence Scores
In Aggregator
The LLM aggregator receives each agent's confidence alongside its A/B preference — explicitly instructed to up-weight high-confidence agent votes when synthesizing the final judgment.
Disagree Flag
If agents disagree and each has high individual confidence, flag the comparison as genuinely ambiguous — down-weight further or surface for human review.
